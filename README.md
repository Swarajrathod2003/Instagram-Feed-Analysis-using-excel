# Instagram Feed Analysis using Excel #
 #Project Overview

This project focuses on analyzing Instagram feed data to uncover insights about engagement rates, post types, and audience preferences. The analysis was conducted entirely in Microsoft Excel using built-in tools — without Power Query — to showcase data cleaning, visualization, and analytical capabilities.

# Objectives

Evaluate engagement performance (likes, comments, reach, etc.)

Identify top-performing post categories and audience trends

Analyze posting frequency and engagement correlation

Build an interactive dashboard for data-driven insights

# Tools & Techniques

Microsoft Excel

Pivot Tables for summarizing engagement metrics

Charts (Bar, Pie, Clustered Column, Line, Scatter) for visual analysis

Slicers & Filters for dynamic data exploration

Conditional Formatting to highlight key trends

# Dashboard Features

Interactive visuals to explore engagement by category and time

Filters to analyze audience behavior based on post type

Trend analysis showing engagement growth over time

# How to Use

Open the Excel file: excel project 1.xlsx

Go to the Dashboard sheet.

Use Slicers and Filters to view engagement data by post type, date, or audience segment.

Explore Pivot Tables for deeper insights and custom analysis.

# Key Insights

Identified the most engaging content types.

Discovered optimal posting frequency for higher reach.

Visualized audience response trends over time.

# Skills Demonstrated

Data Cleaning | Excel Dashboards | Pivot Tables | Data Visualization | Trend Analysis | Analytical Thinking

# Conclusion

The Instagram Feed Analysis project demonstrates the ability to derive actionable insights from social media data using Excel’s advanced analytical tools, helping optimize content strategy and improve engagement performance.
